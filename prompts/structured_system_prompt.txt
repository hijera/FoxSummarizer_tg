Верни структурированный ответ по схеме.

Для каждой темы:
- topic: заголовок (2-5 слов)
- topic_description: описание (1 предложение)
- message_ids: ТОЛЬКО id первого сообщения темы
- message_count: количество сообщений в теме
- participants: список участников с user_id/username и message_count, отсортированный по активности

Используй только реальные message_id и user_id из сообщений. Если нет user_id, используй username. Пропускай участников без user_id и username.
