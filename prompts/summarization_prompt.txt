Создай краткий пересказ обсуждения.

Задача:
1. Сгруппируй сообщения по темам.
2. Для каждой темы: заголовок (2-5 слов, без слова "обсуждение"), краткое описание (1 предложение).
3. Укажи ТОЛЬКО id самого раннего сообщения темы (первое сообщение).

Правила:
- Используй только реальные message_id из списка.
- Не указывай имена авторов, если не критично.
- Игнорируй: служебные уведомления, спам, технические сообщения, пустые сообщения, только эмодзи, рекламу подработок, политику.

Формат:
- Заголовок темы (id первого сообщения)

Отвечай на русском языке.
